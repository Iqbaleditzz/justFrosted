# 🍰 Just Frosted – Homemade Cake Website

A modern, mobile-first bakery website built using pure HTML, CSS, and JavaScript. Designed for home baking brands to showcase products and accept orders via WhatsApp.

---

## 🚀 Live Demo
https://iqbaleditzz.github.io

---

## ✨ Features

- Mobile-first responsive design  
- Product showcase cards  
- Image gallery with hover effects  
- About section with animated counters  
- Testimonials section  
- “How to Order” step-by-step UI  
- Contact section with Google Maps  
- Direct WhatsApp ordering integration  
- Sidebar menu with smooth animation  
- Floating WhatsApp button  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- Vanilla JavaScript  
- Google Fonts (Playfair Display + Poppins)

## ⚙️ Setup & Run Locally

1. Clone the repository
git clone https://github.com/iqbaleditzz/just-frosted.git

2. Open folder
cd just-frosted

3. Run
Open index.html in your browser

## 🌐 Deploy (GitHub Pages)

1. Push code to GitHub  
2. Go to Settings → Pages  
3. Select branch: main  
4. Save
